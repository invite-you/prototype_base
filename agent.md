# AGENT 지침

이 저장소에서 프로토타입을 만들 때 따라야 할 핵심 규칙을 정리했습니다. 모든 작업 전에 이 지침을 확인하세요.

1. **간결성 우선**
   - 코드는 최소한으로 작성하고 복잡도를 줄입니다.
   - 확장성보다 실행 속도와 간결함을 우선합니다.

2. **성능 중시**
   - 지침을 지키는 과정에서 프로그램이 느려지지 않도록 합니다.
   - 불필요한 반복 계산이나 복잡한 추상화를 피합니다.

3. **모델 사용**
   - LLM/임베딩 모델이 필요하면 항상 Hugging Face에서 적절한 모델을 검색해 명시합니다.
   - 모델 이름과 경로를 정확히 기록합니다.

4. **타임스탬프 로그**
   - 작업 단계를 세분화해 각 단계별 설명, 시작/종료 타임스탬프, 소요 시간을 출력합니다.
   - 단계별 로그를 일관된 형식으로 남깁니다.

5. **구성값 관리**
   - 모든 설정값은 전역으로 정의하고, 메인 함수 최상단에 배치합니다.

6. **문서화와 커밋**
   - 주석과 커밋 메시지는 한국어로 상세히 작성합니다.

7. **라이브러리 참고**
   - 특정 라이브러리를 사용할 때는 선택한 버전에 맞는 공식 문서를 함께 확인하고 근거를 명시합니다.
